# Features

## Authentication
- User registration
- User login
- Password hashing with bcryptjs
- Session-based authentication
- Protected routes

## Contact Management
- Add contact
- View contact list
- Edit contact
- Delete contact
- Contact fields: full name, email, phone, company, job title, address, notes, category, status

## Search & Sorting
- Search by name, company, email, or phone
- Filter by category and status
- Sort newest first, oldest first, alphabetical
- Pagination

## Dashboard
- Total contacts count
- Active contacts count
- Follow up contacts count
- Closed contacts count
- Recently added contacts

## UI/UX
- Responsive design for desktop, tablet, mobile
- Modern blue-green theme with glass effect
- Animated buttons and hover states
- Clean forms and tables

## Error Handling
- 404 page
- 500 page
