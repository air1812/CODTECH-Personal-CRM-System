# Personal CRM System

## Project Overview
Personal CRM System is a contact management solution for professionals and learners. It allows users to securely register, manage contacts, and track engagement status using a modern dashboard interface.

## Purpose
Provide a complete full-stack internship project showcasing authentication, CRUD operations, search, pagination, responsive design, and production-ready architecture.

## Key Features
- Secure user authentication with bcrypt hashing
- Session-based route protection
- Contact management with create, read, update, delete workflow
- Search and filter capabilities
- Dashboard stats and recent activity
- Responsive UI with modern glass design

## Target Users
- Sales professionals
- Freelancers
- Startup founders
- CRM learners
