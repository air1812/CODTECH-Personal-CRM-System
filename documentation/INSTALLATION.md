# Installation Guide

## Prerequisites
- Node.js 18 or later
- npm

## Setup Steps
1. Clone the repository.
2. Run `npm install`.
3. Copy `.env.example` to `.env`.
4. Set session secret and environment values in `.env`.
5. Run `npm start`.
7. Open `http://localhost:3001` in your browser.

## Notes
- The database tables are created automatically on the first run.
- Use `npm run dev` during development for live reloading.
